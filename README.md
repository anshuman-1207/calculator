# JavaScript Calculator

A modern and responsive Calculator built using HTML, CSS, and JavaScript as part of the CodSoft Web Development Internship.

## Project Overview

This project is a fully functional calculator that performs basic arithmetic operations through an interactive and user-friendly interface. It is designed to work smoothly across desktop and mobile devices.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Percentage Calculation
- Decimal Support
- Clear (AC) Function
- Delete Last Character
- Keyboard Support
- Responsive Design
- Modern User Interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```text
Calculator/
│── index.html

│── script.js
```
